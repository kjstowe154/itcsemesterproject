Code Sculpt

What is Code Sculpt?

Code Sculpt is a game where you can apply real life coding principles in a fun way to create your own game!

Gameplay

Code Sculpt aims to offer an immersive and educational experience, blending the excitement of game development with the challenges of programming in a virtual environment.
